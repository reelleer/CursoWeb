Javascript

Javascript es un lenguaje dinámico, interpretado y débilmente tipado.

Javascript se usa para maniputar el contenido de un documento, esto puede agregar, eliminar, modificar, mover elementos HTML; agregar, modificar y eliminar estilos.

Para integrar Javascript a nuestro documento hay que hacer uso de de la etiqueta script

<script type="text/javascript">
	alert("Hola mundo!");
</script>

La forma mas recomenda de hacerlo es por medio de un archivo externo

<script type="text/javascript" src="ruta-a-archivo-externo.js" ></script>

Importante: Javascript es sensitivo a minúsculas y mayúsculas.

Referencia: script tag
